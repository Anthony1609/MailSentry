// Project Information

MailSentry AI
{{ dev.project }}

MailSentry AI is an intelligent email threat detection system developed as part of CSC 309 at the Federal University of Technology Owerri. The application uses three supervised machine learning algorithms — Naive Bayes, Logistic Regression, and Support Vector Machine — combined with Natural Language Processing techniques to classify email messages as spam or legitimate with over 98% accuracy. The system features single email analysis, bulk email scanning, a real-time scan history log, and a comprehensive intelligence dashboard.

Algorithms & Tools
Naive Bayes · MultinomialNB Logistic Regression LinearSVC · SVM TF-IDF Vectorizer NLTK · PorterStemmer Scikit-learn Flask · Python 5,572 Training Samples
